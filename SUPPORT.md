# Support

## Getting Help

### Documentation

Full documentation is available in the [`docs/`](docs/) directory:

- [Installation Guide](docs/INSTALLATION.md): setup for Unix, macOS, and Windows
- [Command Reference](docs/COMMANDS.md): all 12 commands with examples
- [Troubleshooting](docs/TROUBLESHOOTING.md): common issues and fixes
- [Architecture](docs/ARCHITECTURE.md): how the skill is structured

### Bug Reports

Found a bug? [Open an issue](https://github.com/AgriciDaniel/claude-blog/issues/new/choose)
using the Bug Report template.

### Feature Requests

Have an idea? [Open a feature request](https://github.com/AgriciDaniel/claude-blog/issues/new/choose)
using the Feature Request template.

### Security Issues

Do **not** open a public issue for security vulnerabilities.
See [SECURITY.md](SECURITY.md) for responsible disclosure instructions.
